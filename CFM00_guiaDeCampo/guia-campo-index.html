¿Qué es esto? El Módulo 00 de CodeFlow es un proyecto estilo escuela 42 para dominar Claude Code: desde /init hasta integrarlo en tus programas via SDK. El subject (el enunciado oficial) te dice qué construir; esta guía te enseña a pensar cada ejercicio — conceptos, estrategia de ataque, trampas frecuentes y las preguntas con las que sabrás si de verdad lo has aprendido.

¿Para quién? Para quien se maneja con terminal, git y código, y quiere pasar de «usar IA» a «controlar un agente». Si vienes de cursos tradicionales, el Campamento Base te explica las reglas del juego antes de que te descoloquen.
